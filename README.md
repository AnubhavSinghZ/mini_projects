# Python Mini Projects
<br>
A collection of beginner-friendly Python mini projects created while learning Python programming.

## Projects Included

### 1. Calculator
<br>
A simple calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

### 2. ATM Simulator
<br>
A basic ATM simulation program that allows users to check balance, deposit money, and withdraw money.

### 3. Number Guessing Game
<br>
A game where the user tries to guess a randomly generated number within a limited number of attempts.

### 4. Snake and Ladders
<br>
A console-based implementation of the classic Snake and Ladders game.
